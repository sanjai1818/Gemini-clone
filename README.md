# Gemini Clone

A feature-rich, front-end clone of Google's Gemini AI interface. Built with modern React and Vite, this application aims to replicate the core user experience of interacting with a conversational AI, including sending messages, viewing a response history, and a clean, minimalist UI.

- **Markdown Support**: AI responses are rendered with basic Markdown formatting (like bold, italics, code blocks).
- **Fast Refresh Development**: Utilizes Vite for a lightning-fast development experience with HMR (Hot Module Replacement).
- **Responsive Design**: Works seamlessly on both desktop and mobile devices.
